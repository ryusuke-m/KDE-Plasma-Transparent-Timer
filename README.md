# KDE-Plasma-Transparent-Timer

KDE Plasma 6 (Wayland) 環境に対応した、ガラスモーフィズムデザインの美しい「透明デスクトップタイマー ＆ アプリ使用時間ロガー」です。

作業時間の記録や、ゲームのしすぎ防止の対策としてデスクトップに常に最前面で表示させることができます。

---

## 🚀 プロジェクトの由来 (Vibe Coding)

このアプリケーションは、Google DeepMind が開発した自律型AIコーディングアシスタント **Antigravity** と、開発者 **ryusuke-m** による **Vibe Coding（バイブコーディング）** の協働によって誕生しました。

Wayland環境における「アクティブウィンドウがセキュリティで保護され直接取得できない」という技術的課題に対し、対話を通じてKWinスクリプトとシステムログ（`journalctl`）を連携させる革新的なソリューションを爆速で導き出し、洗練されたUIへと具現化させました。

---

## ✨ 主な機能

1. **美しいガラスモーフィズムUI**
   - デスクトップの壁紙に溶け込む半透明ぼかし（`backdrop-filter`）とネオンのアクセントカラー。
   - マウスホバー時のみ操作ボタンが浮かび上がるシンプルな設計。
2. **ウィンドウサイズに追従するテキストスケーリング**
   - ウィンドウをマウスで拡大縮小すると、アスペクト比を維持したままタイマーの時間テキストや各種文字サイズが自動的に比例して伸縮します。
3. **アプリ別の自動使用時間計測 (KDE Wayland対応)**
   - 現在アクティブなウィンドウを自動トラッキングし、ウィンドウを切り替えるとリアルタイムで計測対象アプリ名（例: `google-chrome`, `kate`）を検知して表示します。
4. **本日の使用時間可視化**
   - 履歴表示画面から、今日どのアプリをどれだけ使ったかを美しいドーナツグラフ（Chart.js）と比率（%）付きのリストで確認できます。
5. **マウスクリック透過 (Click-Through) モード**
   - 設定画面のトグル、またはグローバルショートカット **`Alt + T`** でクリック透過モードを切り替えられます。
   - 透過中はタイマーの下にあるウィンドウを直接操作でき、作業やゲームの邪魔になりません。
6. **時系列ログデータ設計 (高い拡張性)**
   - 累積時間だけでなく、「いつ、どのアプリを、どれだけ使ったか」を時系列のセッションログ形式（`time_records.json`）で保存しているため、将来的な日次・週次グラフやカレンダー可視化への拡張も容易です。

## 🛠️ セットアップと起動方法

### 前提条件
- Node.js & npm (nvm等)
- KDE Plasma 6 (Wayland) デスクトップ環境

### 1. インストール
リポジトリをクローンしたのち、依存関係をインストールします。
```bash
cd KDE-Plasma-Transparent-Timer
npm install
```

### 2. 起動
```bash
npm start
```
※ Linux環境でのサンドボックスの競合を防ぐため、自動的に `--no-sandbox` フラグを付与して起動するようパッケージングされています。

### 3. 操作方法
- **移動**: ウィンドウ上部のドラッグエリアをつまんで自由に配置できます。
- **コントロール**: タイマーをホバーすると現れるボタンから「開始」「一時停止」「リセット」が可能です。
- **クリック透過の解除**: 透過モード中はキーボードの **`Alt + T`** を押すことで通常モードに復帰します。
- **終了**: ホバー時に右上に表示される **「×」** ボタンから終了します（終了時にKWinスクリプトの解除とセッションの保存が自動で行われます）。
